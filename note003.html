Note 03 - Downloading and installing nodejs on a linux machine

Step 01update internal list of software

  • Open terminal in Linux and...
    • sudo apt-get -y update

Step 02check whether you have installed curl

  • type in the terminal
    • sudo apt-get install curl

    curl is a tool that can be used to download things from internet on your machine

Step 03adding nodejs repository to your machines apt-get repository

  • Go to Nodejs web page
  • click on downloads
  • click on installing Nodejs via package manger
  • click on Debian and Ubuntu based Linux distributions
  • copy paste the commands shown in the code block in to your command line
    • curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -

Step 04Installing node and npm

  • type in the terminal
    • sudo apt-get install -y nodejs

Step 05check npm version

  • type in the terminal
    • npm --version

Step 06check node version

  • type in the terminal
    • node version
